# Rays
Запуск:
1) Через командную строку 
  -  mkdir build
  -  cd build
  -  cmake ..
  -  make
  -  ./tinyraytracer
 
2) Через IDE (Clion например)
    - open project
    - подгрузить cmake
    - run 
    
Результат: на выходе в папке build изображение out.ppm с отрисованной сценой.
